# Edwiser Bridge - Course Experience

An Edwiser Bridge extension that provides a seamless course viewing experience within WordPress by integrating with Moodle.

## Requirements

- WordPress 6.0+
- PHP 7.4+
- Edwiser Bridge plugin

## Usage

This plugin extends Edwiser Bridge and uses its API token for Moodle integration. No additional configuration is required.
